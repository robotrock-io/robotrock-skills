# RobotRock agent skills

Public agent skills for integrating [RobotRock](https://robotrock.io) human-in-the-loop approvals.

## Install

```bash
npx skills add robotrock-io/robotrock-skills --skill robotrock
```

Registry: [skills.sh/robotrock-io/robotrock-skills/robotrock](https://skills.sh/robotrock-io/robotrock-skills/robotrock)

## Contents

| Skill | Description |
|-------|-------------|
| `robotrock` | SDK integration — sendToHuman, webhooks, Trigger.dev, Vercel Workflow, AI SDK, Eve agents |

## SDK

npm: [`robotrock`](https://www.npmjs.com/package/robotrock) — source mirror at [robotrock-io/robotrock-sdk](https://github.com/robotrock-io/robotrock-sdk)
